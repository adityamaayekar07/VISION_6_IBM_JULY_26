# Vision 6 AI

## Overview

Vision 6 AI is an AI-powered Exam Preparation Assistant designed to help students learn smarter, practice better, and succeed faster.

The application provides an interactive interface for study assistance, note summarization, quiz generation, MCQ generation, interview preparation, and AI-powered learning support.

---

## Features Implemented

### Dashboard

* Modern dashboard layout
* Statistics cards
* Quick access feature cards
* Welcome banner

### AI Chat Interface

* User and AI chat messages
* Chat window UI
* Streaming response animation
* Typing indicator
* Auto-scroll functionality
* Enter key support

### Study Tools

* Topic explanation interface
* Notes summarizer interface
* MCQ generator interface
* Quiz generator interface
* Interview preparation interface

### Data Management

* Local Storage integration
